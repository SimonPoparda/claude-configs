# Set-up the repo
The best way is to clone it to the separate folder and then copy and paste the necessary files to the other projects OR:

## download as zip to another project:
curl -L https://github.com/SimonPoparda/claude-configs/archive/refs/heads/main.zip -o claude-configs.zip && unzip claude-configs.zip && rm claude-configs.zip
